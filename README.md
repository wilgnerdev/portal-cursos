## Sobre o Projeto

Este portal foi desenvolvido como parte do **Trabalho de Conclusão de Curso (TCC)**
do curso de aprendizagem profissional Suporte de TI na Fundação O Pão dos Pobres, o qual eu sou aluno.

O objetivo é oferecer uma interface responsiva e acessível para que alunos, responsáveis
e interessados possam visualizar os cursos disponíveis na instituição.

## 🚀 Como Executar Localmente

1. Clone o repositório:
```bash
git clone https://github.com/wilgnerdev/portal-cursos.git
```

2. Abra o arquivo `index.html` no seu navegador.

> Não é necessário nenhuma instalação adicional.
